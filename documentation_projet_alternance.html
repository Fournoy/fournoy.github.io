Outil d’automatisation SI – Projet d’alternance

Objectif général

Durant mon alternance, certaines opérations sur le domaine Active Directory étaient longues, répétitives ou limitées par les outils natifs Windows.

L’objectif du projet était de développer un outil centralisant plusieurs scripts afin de faciliter et d’accélérer les tâches quotidiennes des techniciens SI.

Fonctionnalités apportées

  • Automatisation de tâches récurrentes : redémarrage programmé à distance

  • Adaptation d’USMT pour créer des sauvegardes utilisateurs personnalisées

  • Scripts d’inventaire Active Directory : groupes, sous-groupes, utilisateurs inactifs depuis X jours…

Spécificités techniques

L’environnement imposait l’utilisation exclusive de technologies natives Windows.

Interface graphique : développement en C++ avec la bibliothèque windows.h (WinAPI).
Scripts : PowerShell et Batch pour l’automatisation des tâches.

L’outil a été conçu pour s’intégrer sans dépendances externes dans l’infrastructure existante.
