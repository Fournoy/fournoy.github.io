PuzzleSQL üíâ

Package to automate your SQL injection

PuzzleSQL is a Python package designed to simplify and speed up the automation of SQL injections (SQLi). It offers pre-built functions that users can assemble like puzzle pieces to create custom scripts.

GitHub Repository

Main Features

  • SQL Injection Automation: Provides functions to automate time-based and error-based SQL injections
  • Binary Search Algorithm: Uses binary search (O(log n)) instead of linear search for faster data extraction
  • Customization: Users can assemble functions like puzzle pieces to create scripts tailored to their needs
  • Evolving Support: Future support for WAF bypass techniques and out-of-band SQL injection with DNS/HTTP server

Binary Search Advantage

PuzzleSQL uses binary search which provides O(log n) complexity compared to linear search's O(n). This is especially beneficial for Blind SQLi when extracting passwords or information:

  • Linear search (n): Tries every possible value
  • Binary search (log n): Divides data in half each iteration, finding answer exponentially faster

Binary Search Function

def binary_search_boolean(result_bool: bool, operator1: str, 
                           hight: int, low: int, binary_average: int) -> int:
    if result_bool == True and operator1 == '<':
        hight = binary_average - 1
    if result_bool == True and operator1 == '>':
        low = binary_average + 1
    if result_bool == False and operator1 == '>':
        hight = binary_average - 1
    if result_bool == False and operator1 == '<':
        low = binary_average + 1
    return hight, low, binary_average

This function refines the boundaries of a binary search based on SQL injection test results, helping determine the correct numerical value.
